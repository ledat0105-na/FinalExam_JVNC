Import sản phẩm từ CSV

Tải lên file CSV để import sản phẩm vào hệ thống.

Hướng dẫn định dạng CSV

File CSV cần có các cột theo thứ tự:

  1. SKU (bắt buộc)
  2. Item Name (bắt buộc)
  3. Category ID (bắt buộc)
  4. Item Type (PRODUCT hoặc SERVICE)
  5. Unit Name
  6. Unit Price
  7. Weight (Kg) - tùy chọn
  8. Description - tùy chọn
